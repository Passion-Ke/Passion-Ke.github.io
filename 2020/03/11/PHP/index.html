PHP的工作原理是什么？阐述WAPM的含义以及PHP的工作原理。

由Web服务器和PHP解释器解释完成，Web服务器主要使用Apache。

？WAPM：

  • W => Windows
  • A => Apache
  • P => PHP
  • M => MySQL

当用户在浏览器中输入PHP页面地址，触发一个web请求，并将请求转发到web服务器（Apache）。

web服务器接受请求，根据后缀名判断为一个PHP请求（如非PHP请求 如html JS CSS等静态资源 web服务器直接处理返回给浏览器），web服务器调出PHP应用程序转发到PHP解释器。

PHP解释器将web服务器转发来的PHP文件处理后，动态生成html页面。

PHP解释器将生成的html页面返回给web服务器，web服务器将html页面返回给客户端浏览器展示给用户
